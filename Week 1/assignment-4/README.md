# AI vs Traditional Software Report

## AI-Based Apps

1. ChatGPT – Helps generate responses and answers questions
2. Google Assistant – Responds to voice commands and questions
3. Siri – Helps perform tasks using voice
4. Netflix – Suggests movies and shows based on what you watch
5. Spotify – Recommends music based on your listening habits

## Traditional Software

6. Calculator – Performs fixed calculations
7. Notepad – Used for typing simple text
8. Microsoft Word – Used to create and edit documents
9. Paint – Used for basic drawing
10. File Explorer – Helps manage files and folders
11. Clock App – Used for time and alarms
12. PDF Reader – Used to open and read documents
13. Camera App – Takes photos without learning behavior
14. Contacts App – Stores and displays contact information
15. Calendar – Used for scheduling events

## Conclusion

Some applications are able to adapt and give suggestions based on user behavior, while others simply perform fixed tasks without changing. This shows the difference between systems that respond dynamically and those that follow set instructions.
